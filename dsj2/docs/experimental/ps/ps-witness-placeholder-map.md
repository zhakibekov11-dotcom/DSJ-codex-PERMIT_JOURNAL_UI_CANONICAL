# ПС Свидетельство: коды шаблона

Шаблон [ps-witness-certificate-template.docx](/Users/macm2/Documents/Новая папка/DSJ/dsj2/docs/experimental/ps/ps-witness-certificate-template.docx) переведен на явные коды, чтобы дальнейшая автоматизация не зависела от расплывчатых `(дата)` / `(год)`.

Основные коды:

- `{{KB_NUMBER}}` — номер КБ целиком, например `КБ № 26/ПС/СВ-000001`
- `{{REGISTRATION_NUMBER}}` — регистрационный номер свидетельства, например `00001`
- `{{FULL_NAME_RU}}` — ФИО на русском
- `{{FULL_NAME_KZ}}` — в текущем этапе повторяет то же ФИО, что и русская сторона
- `{{PROFESSION_RU}}` — профессия на русском
- `{{PROFESSION_KZ}}` — профессия на казахском
- `{{EDU_ORG_RU}}` — организация обучения на русском
- `{{EDU_ORG_KZ}}` — организация обучения на казахском
- `{{PROTOCOL_NUMBER_DISPLAY}}` — номер протокола в формате ПС

Даты обучения и выдачи:

- `{{TRAINING_START_DAY}}`
- `{{TRAINING_START_MONTH_RU}}`
- `{{TRAINING_START_MONTH_KZ}}`
- `{{TRAINING_START_YEAR_SHORT}}`
- `{{TRAINING_START_YEAR_FULL}}`
- `{{TRAINING_END_DAY}}`
- `{{TRAINING_END_MONTH_RU}}`
- `{{TRAINING_END_MONTH_KZ}}`
- `{{TRAINING_END_YEAR_SHORT}}`
- `{{TRAINING_END_YEAR_FULL}}`
- `{{ISSUE_DAY}}`
- `{{ISSUE_MONTH_RU}}`
- `{{ISSUE_MONTH_KZ}}`
- `{{ISSUE_YEAR_SHORT}}`

Текущая база для генерации:

- дата окончания обучения по умолчанию равна дате выдачи
- дата начала обучения по умолчанию считается как `дата выдачи - 4 дня`
- длительность по умолчанию — `5` дней
