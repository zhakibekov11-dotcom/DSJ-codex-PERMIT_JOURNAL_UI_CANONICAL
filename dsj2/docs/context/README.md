# Context Navigation

This folder is the human navigation layer of the Repo Context Operating System.

Use it to orient quickly. Do not treat it as the product source of truth.

## Read First

- `start-here-for-codex.md`
- `current-product-status.md`
- `repo-context-operating-system.md`
- `execution-phases-p0-p4.md`

## Routing Helpers

- `CONTEXT_QUERY_INDEX.md` - lightweight query router for humans and Codex
- `FEATURE_BRANCH_MATRIX.md` - branch and feature routing metadata
- `branches/` - branch-specific execution context documents
- `.mempalace/index.json` - machine-readable repo router
- nearest area `AGENTS.md` - local rules for the touched files

## What This Folder Is For

- fast repo orientation in a new chat
- current product status snapshots
- branch phase definitions
- pointers to the canonical files that matter before code changes
- area/package README files such as `apps/web/README.md` and `packages/database/README.md`

## What This Folder Is Not

- not the product architecture itself
- not a replacement for code
- not a replacement for package scripts or area AGENTS
